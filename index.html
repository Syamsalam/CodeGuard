Detection Process

1 AST Tokenization

Input: Source code files (.py, .js)

Process: Parse code into Abstract Syntax Tree and extract structural tokens

Output: Token sequences representing code structure

Example tokens:
['NODE_FunctionDef', 'FUNC_DEF', 'NODE_If', 'NODE_Compare', 'NODE_Return', ...]
2 TF-IDF Vectorization

Formula: tfidf(t,d,D) = tf(t,d) × idf(t,D)

Process: Convert token sequences to numerical vectors

Output: High-dimensional sparse matrices

TF-IDF Matrix:
Document 1: [0.45, 0.23, 0.00, 0.67, ...]
Document 2: [0.32, 0.56, 0.12, 0.45, ...]
3 Cosine Similarity

Formula: cos(θ) = (A·B) / (||A|| × ||B||)

Process: Calculate pairwise similarities between all documents

Output: Similarity matrix with scores 0-1

Example: 0.75 similarity score (75%)
4 Classification & Reporting

Threshold: Configurable similarity threshold (default: 0.7)

Classification: Similarities above threshold marked as plagiarism

Output: Comprehensive reports with visualizations
